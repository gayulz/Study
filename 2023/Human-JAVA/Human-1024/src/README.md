
# 일정관리 프로그램
-[ ] 기본 메뉴는 일정등록, 수정, 삭제, 전체보기가 있다.
-[ ] 일정을 등록 할 때는 제목, 내용, 중요도가 있다.
-[ ] 수정할때는 제목으로 검색 후 수정한다. 내용과 중요도만 수정할 수 있다.
-[ ] 삭제는 제목으로 검색 후 삭제 한다.
-[ ] 젠체보기는 모든 일정이 보여진다.

# 업그레이드 1
- 검색하기 기능 추가 : 제목과 정확하게 일치 하는 일정만 검색한다.
- 일정등록 기능 업그레이드 : 특수문자는 포함되지 않도록 유효성을 체크 한다.
- 특수문자 체크는 별도로 메서드를 만든다
- 삭제하기 기능 업그레이드 : 정말로 삭제 하겠습니까? 라는 확인 메시지 추가

# 업그레이드2
- 전체보기 기능 업그레이드 : 중요도가 높은 것에서 낮은 것으로 정렬하여 출력한다.
- 관심 일정 관리 : 일정 중  관심도가 높은 일정은 별도로 체크 한다.
- 전체보기에서는 관심일정이라는 표시로 제목옆에 * 추가한다.
- 관심일정보기 메뉴를 추가한다.
- 검색하기 기능에서 관심일정으로 지정할 수 있다.
