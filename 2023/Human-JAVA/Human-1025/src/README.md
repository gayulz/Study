# 과제 : 영화 예매 프로그램
**팀원 : 장민지/김가율**

택 1 은 난이도를 조별로 선택하세요

✅ 메뉴는 **예매, 예매확인, 전체보기, 정보수정, 취소** 가 있다

### 📌 필수기능
**좌석** 
- 선택 : 좌석은 10개가 있다
- 선택 : 좌석은 1열,2열,3열이 있고 1열 6개, 2열 8개, 3열 10개

**예매시 이름과 비밀번호를 기록한다**

**예매 확인은 이름을 입력하면 예약된 좌석번호와 이름을 확인한다**

**전체보기**
- 선택 : 좌석별 예매한 사람의 이름을 출력한다
- 선택 : 좌석별 예약, 예약기능이 출력되고 예약일 경우 예약자 성 뒤에 마스킹처리후 출력

**수정은 예매자 이름으로 수정 가능하고 비밀번호만 수정 가능하다**

**삭제는 예매자 이름으로 검색하여 예매를 취소한다**

\
✏️ 추가기능1
- 선택 : 예매시 동일 이름으로는 예약 불가하다
- 선택 : 이름에 특수문자가 포함되면 안된다.

\
✏️ 추가기능2 
- 자리 이동 기능을 추가한다
- 빈 자리를 출력하여 원하는 자리를 선택하고 원하는 자리가 있으면 비워서 이동시킨다
- 빈 자리가 아닐 경우 거절한다

\
✏️ 추가기능3
- 선택1 : 회원제로 변경하고 로그인 후 예매시스템을 이용하도록한다
- 선택2 : 회원제로 변경하고 메뉴 중 전체보기는 로그인을 안해도 되나, 다른 메뉴는 로그인 후 이용 가능하다

# 시나리오 1
로그인을 한다
- 로그인 할때엔 회원가입과 로그인중 선택해야한다
- 로그인 3회 오류시 프로그램종료
- 로그인 정상 일 경우 로그인 한 회원의 이름 출력화 함께 예매 프로그램으로 넘어간다

로그인 성공 후 
- 메뉴를 보여준다
- 메뉴는 예매/예매확인/전체보기/정보수정/취소 가 있다

회원가입
- 이름과 비밀번호를 입력한다

# 시나리오 2
예매 메뉴
- 동일 이름으로 예약한 건이 있는지 확인한다
- 동일 이름으로 예약한 건이 이미 있는 경우 : 불가하다는 출력과 메뉴로 이동
- 동일 이름 예약건 없는 경우 예약가능한 자리를 보여준다
- 예약 인원을 입력한다
- 그만큼 좌석을 선택하게한다
- 완료 후 좌석 번호와 갯수를 출력한다

예매 확인
- 예매한 사람의 이름을 입력한다
- 예매된 좌석 번호와 이름을 출력한다 

전체 보기
- 전체 좌석의 정보를 보여준다 . 예약자의 이름은 성 빼고 마스킹처리한다

예매좌석 수정
- 예매한 사람의 이름을 입력한다
- 없는 경우 없다고 출력한다
- 있는 경우 예약한 인원 수 와 예약한 좌석 번호를 보여준다
- 예매 가능한 좌석을 출력한다
- 선택한 자리로 변경한다 
- 변경한 내용을 출력해준다

예매 취소
- 예매자 이름을 입력한다
- 없는 경우 예약자 정보가 없다고 출력
- 있는 경우 예약자 이름과 인원수 좌석번호를 출력한다
- 취소할것인지 재차 물어본다
- 취소시 즉시 취소되고 메뉴로 복구된다
- 취소를 하지 않는다고 할 경우 취소 안되었다며 메뉴로 복구

기능
- 이름은 특수문자를 넣어선 안된다
- 기존 좌석에 예약이 있는지 없는지 판단하고 리턴해주는게 필요하다
