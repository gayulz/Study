스티# 주말 스터디
### 오늘의 주제 : 팩토리 패턴

- 객체 생성을 공장(Factory)클래스로 캡슐화 처리한 다음 대신 생성하게 하는 생성 디자인 패턴이다
- 클라이언트에서 직접 new 연산자를 통해 생성하는 것이 아닌 객체들을 도맡아 생성하는 공장 클래스를 만들고 이를 상속하는 서브 공장 클래스의 메서드에서 여러가지 제품 객체 생성을 각각 책임지는 것 이다
- 객체 생성에 필요한 과정을 템플릿 처럼 미리 구성하고, 객체 생성에 관한 전처리나 후처리를 통해 생성 과정을 다양하게 처리하며 객체를 유연하게 정할 수 있는 특징이 있다
